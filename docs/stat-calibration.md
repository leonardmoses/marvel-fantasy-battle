# Stat Calibration Guide

This file defines the measurement framework and anchor points for each of the 25 stat categories. Use this as a reference when assigning stats to characters to ensure consistency across the roster.

**General Rules:**
- Scale is 1–10 for most stats. 0 = ability is completely absent (not low — does not exist).
- Average superhero/mutant baseline = 3
- Scale is non-linear — gaps between numbers reflect real power differences, not equal increments
- Ceiling-breakers (above 10) are reserved for characters who genuinely transcend the normal scale
